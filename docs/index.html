React wheel jank demo

Demo of React wheel jank caused by attaching a global wheel event listener to the document, even when we only really need to listen on the wheel events for an inner div.
  • Version 1: No wheel events at all
  • Version 2: React onWheel in the inner div
  • Version 3: Actual onwheel event in the inner div
  • Version 4: React onTouchStart event in the inner div
  • Version 5: Actual ontouchstart event in the inner div
